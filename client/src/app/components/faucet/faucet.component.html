Token Faucet

Distribution of these tokens ( 0x24ddd7c47d5ea7181f4d3bd67fb1d361c49020c6 ) are for development purposes only. The crowdfunding contract allows users to choose the token that they want to be funded in. Contract will check if the provided address is not a contract and contains supply of more than 0. Since this application is deployed on Ethereum Sepolia, a token was created to facilitate development. The faucet distributes 1000 DEV tokens each time and has a 25 hours cooldown per address.
